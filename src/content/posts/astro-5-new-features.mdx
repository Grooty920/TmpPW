## 前言

Astro 5 于近期正式发布，作为一个已经在生产环境中使用 Astro 的开发者，自然要第一时间体验新版本的变化。

## 内容层 (Content Layer)

Astro 5 最重磅的更新是全新的 **Content Layer** 系统，它让内容管理从"约定"走向了"标准"：

```typescript
import { defineCollection, z } from "astro:content";

const posts = defineCollection({
  type: "content",
  schema: z.object({
    title: z.string(),
    description: z.string(),
    date: z.date(),
    tags: z.array(z.string()),
  }),
});
```

## 更快的构建速度

根据官方基准测试，Astro 5 的构建速度提升了约 **30%**，这得益于 Vite 6 的底层升级和更智能的静态分析。

## 升级指南

从 Astro 4 升级到 5 非常简单：

```bash
npx @astrojs/upgrade
```

## 小结

如果你还没尝试过 Astro，现在是最好的时机。
